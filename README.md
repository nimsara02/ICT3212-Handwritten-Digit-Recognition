# Handwritten Digit Recognition - ICT 3212

## Project Overview
This repository contains Phase 03 - Implementation 1 of our Image Classification project. We are using the MNIST dataset to recognize handwritten digits (0-9).

## Implementation 1 (Baseline)
- **Model Type:** Simple Dense Neural Network (Non-CNN)
- **Framework:** TensorFlow / Keras
- **Dataset:** MNIST
- **Accuracy Achieved:** 97.32% (approx)

## Files included:
- `Implementation_01_Code.ipynb`: The Google Colab notebook containing code and graphs.
- `handwritten_baseline_model.h5`: The saved baseline model file.

## Implementation 2 - Improvements
- **Architecture**: Upgraded from a Dense Baseline to a Convolutional Neural Network (CNN).
- **Techniques**: Added Dropout (0.5) and Early Stopping to reduce overfitting.
- **Result**: Accuracy improved from 97.32% to over 99%.
